registerToModelFunction

  • registerToModelFunction(type: string, createModelFunctionCreator: CreateModelFunctionCreator): void
    • Defined in packages/ckeditor5-coremedia-content-clipboard/src/ContentClipboardEditing.ts:59

    This function is used to register "toModel" functions in other plugins. These functions are held in the ContentToModelRegistry and are used to insert dropped content into the editor.

    Please note: Types that are not supported by the DataToModelMechanism will fall back to the default "toModel" function or throw an error.

    Parameters

    • type: string

      the identifier for the dropped content (e.g. "link" or "image")

    • createModelFunctionCreator: CreateModelFunctionCreator

      a function that expects a contentUri as parameter and returns a promise of type CreateModelFunction

    Returns void
