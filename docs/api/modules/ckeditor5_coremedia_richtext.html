Module ckeditor5-coremedia-richtext

This plugin is required to edit CoreMedia Richtext 1.0, an XML format, which provides a subset of XHTML features.

It grants conversion of CoreMedia RichText 1.0 to the CKEditor data model as well as conversion from CKEditor data model to CoreMedia RichText.
