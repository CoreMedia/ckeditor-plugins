langMapper: AttributeMapper = ...

Maps xml:lang and lang from data (CoreMedia RichText) to lang. Just as in the standard definition, xml:lang is preferred over lang when both are set in CoreMedia RichText.

Note, that on transformation to data, the language will always be stored in xml:lang.
