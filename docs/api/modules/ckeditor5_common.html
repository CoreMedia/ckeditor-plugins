Const ifPlugin

  • ifPlugin<T>(editor: default, key: PluginInterface<T>): Promise<T>
    • Defined in packages/ckeditor5-common/src/Plugins.ts:95

    Promise, which either resolves immediately to the given plugin or rejects with Error if not available.

    If you refer to a required plugin, skipping catch for the promise may be fine. For optional plugins (trigger an action if plugin is available) you may want to use optionalPluginNotFound as handler, which just logs a debug note on a not existing plugin.

    example
    ifPlugin(editor, OptionalPlugin)
      .then(...)
      .catch(optionalPluginNotFound);
    throws

    PluginNotFoundError if plugin could not be found

    Type parameters

    • T: default<T>

    Parameters

    • editor: default

      editor to find requested plugin

    • key: PluginInterface<T>

      plugin key

    Returns Promise<T>

    Promise for requested plugin
