CKEditor 5: CoreMedia Plugin Showcase

CKEditor 5: CoreMedia Plugin Showcase

This example instance of CKEditor 5 serves as a showcase for plugins provided by CoreMedia. Most of these plugins are mandatory to use CKEditor 5 as editor within CoreMedia Studio. Others are required to edit CoreMedia RichText, and then again others provide more additional functionality.

Mandatory Plugins for CoreMedia Studio

Not existing yet, but they will for example provide hooks to synchronize CoreMedia Studio and CKEditor.

Required Plugins for CoreMedia Richtext

These plugins are required to edit CoreMedia Richtext within CoreMedia Studio. They will especially convert HTML to CoreMedia Richtext and vice versa.

Optional Plugins

There are a bunch of plugins which are used by our customers to further extend the functionality of CKEditor. You don't need to add them in order to use CKEditor within CoreMedia Studio or as CoreMedia Richtext editor.

SymbolOnPasteMapper

The initial and main purpose of this plugin is to replace characters pasted from Microsoft Word with font _Symbol_ to their corresponding Unicode characters. This refers to older versions of Microsoft Word, typically. Nowadays, Microsoft Word itself often inserts Unicode characters directly.

CoreMedia RichText

This plugin will transform HTML in CKEditor to CoreMedia RichText 1.0 and vice versa. Here are some elements, which require special notice:
